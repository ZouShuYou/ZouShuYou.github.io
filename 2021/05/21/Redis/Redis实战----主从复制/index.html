这里使用 docker-compose 来测试 Redis 的主从复制功能。

首先需要安装 docker 和 docker-compose 以及 Redis 镜像，这里就不再做这方面的安装介绍了。

安装好后，需要编写 docker-compose.yml 文件

然后在当前文件夹下，输入命令: docker-compose up -d

测试主从复制：

  1. 测试是否能向从库写入数据？
  2. 测试写入数据到主库，从库有没有数据？
  3. 测试一个从库节点挂掉，主库写入数据，从库节点重启后，是否能同步主库数据？
  4. 测试主库挂点，从库是否能读取数据？

答案：

  1. 不能
  2. 有
  3. 能同步
  4. 能
